# Assignment 1

## Problem Statement

Create a simple HTML page that demonstrates commonly used HTML text formatting tags in a clear and structured format.

## Objective

- Understand basic HTML page structure
- Learn how different text formatting tags work
- Present the tags, their purpose, and examples in a readable table

## Explanation

This assignment is built using plain HTML and internal CSS.

- The page displays a heading named `HTML Text Formatting Tags`
- A table is used to organize the content into three columns:
  - Tag
  - Description
  - Example
- The examples show how tags like `<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, `<sup>`, and `<u>` appear in the browser
- Basic styling is added to improve readability using background colors, borders, spacing, and alternating row colors

## Output

- A clean webpage containing a table of HTML formatting tags
- Each row shows the tag name, its description, and a live example of the formatted text
- The output helps beginners visually understand the effect of each formatting tag
