# Assignment 7 - Student Feedback Review System

This project is a simple full-stack web application made with React.js, Express.js, and MongoDB.

## Folder Structure

- `frontend` - React user interface
- `backend` - Express server with MongoDB connection

## Backend Setup

1. Open the `backend` folder.
2. Run `npm install`.
3. Create a `.env` file by copying `.env.example`.
4. Add your MongoDB connection string in `MONGODB_URI`.
5. Run `npm run dev`.

## Frontend Setup

1. Open the `frontend` folder.
2. Run `npm install`.
3. Run `npm run dev`.

## Features

- Submit student feedback
- Store feedback in MongoDB
- View all submitted reviews
- See total reviews and average rating
- Clean and simple student-friendly UI
