# Assignment 2

## Problem Statement

Design and develop a personal portfolio webpage using HTML and CSS that introduces the student, highlights skills and projects, and shares contact information.

## Objective

- Build a multi-section portfolio website
- Practice page layout, navigation, and styling with CSS
- Present personal, academic, and project information in a professional format

## Explanation

This assignment is a static portfolio website created with HTML and CSS.

- A navigation bar links to the main sections: About, Skills, Projects, and Contact
- The hero section introduces Tanyya Balwir with a profile image, title, and social links
- The About section gives a short personal introduction
- The Skills section shows technical skills such as HTML, CSS, JavaScript, MongoDB, React, Node.js, Python, and SQL
- The Projects section presents multiple project cards with short descriptions
- The Contact section displays the email address for communication
- Custom styling in `style.css` is used to make the page visually appealing and well-structured

## Output

- A responsive portfolio-style webpage with smooth section-based navigation
- A personal profile area with image and social media links
- Clearly organized sections for skills, projects, and contact details
