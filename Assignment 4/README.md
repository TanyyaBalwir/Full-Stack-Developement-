# Assignment 4

## Problem Statement

Create a simple sales analytics dashboard that visualizes business data using charts and allows interactive data updates.

## Objective

- Learn how to integrate Chart.js into a web page
- Present different types of data using multiple chart formats
- Add interactivity by updating chart values dynamically

## Explanation

This assignment is a frontend dashboard built using HTML, CSS, JavaScript, and Chart.js.

- A bar chart is used to display monthly sales data
- A doughnut chart is used to show product category distribution
- A line chart is used to represent weekly revenue trends
- The `Update Data` button generates new random values and refreshes all three charts
- The page layout groups the charts into a dashboard format for easy viewing

## Output

- A sales dashboard with three visual charts:
  - Monthly Sales
  - Product Categories
  - Weekly Revenue Trend
- An interactive button that updates all charts with new data in real time
